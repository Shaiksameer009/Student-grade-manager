# Student-grade-manager
A command-line interface (CLI) application for basic student data management using file handling. This project implements core functionalities like creating, reading, and appending student records, along with a logging feature. 

#Simple Student Data Manager

This Python-based project provides a simple system to manage student records via file I/O operations. It is a console-based application, designed for beginners to practice fundamental programming concepts like data structures and file handling. 

#Features
The system is organized around three main tasks: 
TASK 1: Create and Write Student Data: Allows users to input new student details and write them to a data file.
TASK 2: Read and Display Data: Reads the stored student records from the file and displays them in a clear, formatted manner.
TASK 3: Append New Student and Create Log: Enables appending new student data to the existing file without overwriting old records and generates a log of the activity. 
Technologies Used:
    -Python 3.x
